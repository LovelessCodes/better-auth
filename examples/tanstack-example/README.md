An example of using Better Auth with [TanStack Start](https://tanstack.com/start).

## Setup

To install dependencies:

```bash
pnpm install
```

To migrate Better-Auth:

```bash
pnpx @better-auth/cli migrate
```

To run:

```bash
pnpm dev
```
